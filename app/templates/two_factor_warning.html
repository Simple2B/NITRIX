2-Step Verification Setup

To set up two-factor authentication, Google Authenticator applicaion is required. You can download it using following links:

Follow this instructions to complete the set up
  1. Open Google Authenticator application on your mobile device
  2. Press “ + “ button at the top-right corner of the screen
  3. Select “Scan barcode” option from the pop-up menu
  4. Scan QR-code which will appear on the next page with your mobile device
  5. Verify your username under the 6-digit OTP code generated in Google Authenticator application

SECURITY ALERT :The following page contains sensitive information. Make sure you are not sharing your screen to anybody while setting up two-factor authentication.
